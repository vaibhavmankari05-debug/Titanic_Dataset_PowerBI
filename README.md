**Overeview :**

Titanic Survival Analysis in Power BI This project presents an exploratory data analysis (EDA) of the famous Titanic dataset using Power BI to uncover factors that influenced passenger survival.

üéØ Project Goal:
 The primary goal of this analysis is to:Analyze the survival rates of Titanic passengers.Identify key demographic and socio-economic factors (like passenger class, gender, and age) that correlate with survival.Visualize the insights using an interactive Power BI dashboard.

üìä Key Findings (Illustrative Examples):
 The Power BI dashboard highlights several crucial insights:Gender: Females had a significantly higher survival rate than males, reflecting the "women and children first" protocol.Passenger Class (Pclass): First-class passengers generally had the highest survival rate, while Third-class passengers had the lowest, suggesting the impact of location and priority access to lifeboats.Age: Children (Age 0-9) often exhibited a better survival rate than adults, also aligning with the emergency protocol.
